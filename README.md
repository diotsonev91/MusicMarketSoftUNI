# MusicMarketSoftUNI
A marketplace web application for musicians to buy and sell used musical instruments and gear. Built with Angular and Firebase (AngularFire) for authentication and data management.

## Project Overview
This is a marketplace web application for musicians to buy and sell second-hand instruments and music equipment, built with **Angular (v18.2.10)** and **Firebase**. Inspired by OLX but focused on the music community, this project allows users to post ads, browse listings, and connect with other musicians.

## Folder Structure
The project is organized into two main folders:

- **Frontend/**: Contains the Angular application code for the user interface and client-side logic.
- **Backend/**: Contains Firebase configurations and setup files needed to run Firebase services (authentication, Firestore, storage, etc.) when presenting on the exam.

## Features (Planned)
- **Public Part**: Browse ads without login
- **Private Part**: Post, edit, and manage ads (for registered users)
- **Search and Filter**: Find specific instruments or equipment
- **User Profiles**: View and manage personal ads, connect with other users

## Tech Stack
- **Frontend**: Angular (v18.2.10)
- **Backend**: Firebase (using AngularFire for integration)
  - **Firebase Authentication**: For user registration and login
  - **Firestore Database**: For storing ad listings and user data
  - **Firebase Storage** : For handling image uploads
